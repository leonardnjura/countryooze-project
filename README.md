# countryooze-project
The public version of a project in NextJS. Built for travelers and territory scholars

#### How should this be manually tested?
- setup next.js project, 
- run `$ git clone <repo>` and `$ cd <branch>`
- run `$ yarn install` 
- run `$ yarn dev`

- go to localhost:3000
